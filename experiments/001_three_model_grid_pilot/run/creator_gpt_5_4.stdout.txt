Built the package in place as `Occluded Tile Provenance`: a 30-item raster visual benchmark where four same-texture cardboard pieces overlap, and the solver must identify which badge-labeled piece owns the surface under a red crosshair. The public bundle contains only item JSONL plus relative PPM images; gold answers stay separate.

Validation is complete. The required generator command produced all 30 items, `verifier.py` passed on the solver bundle against gold, gold-on-gold scoring returned `30/30`, and a weak constant-`C` baseline scored `9/30` (`0.30`). I also checked the solver bundle for leakage and found no gold, seeds, hidden geometry, or scorer/generator logic in the public inputs; only benign schema/example text mentions. The only runtime oddity was a harmless `pyenv` rehash warning before Python commands.
